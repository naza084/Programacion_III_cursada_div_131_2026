# Atajos de HTML (Emmet)


| Atajo | Descripción |
|-------|-------------|
| `div*3` | Crea 3 divs |
| `div>h3+p` | Crea un div con un h3 y un p dentro |
| `div#id` | Crea un div con un id específico |
| `div.class` | Crea un div con una clase específica |
| `div#id.class` | Crea un div con un id y una clase específica |

## Ejemplos

- `div#estero>h3+p`: Crea un div con id "estero", con un h3 y un p dentro
