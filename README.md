## Temario de la materia

- **HTML5 semántico:** Estructura de páginas web, formularios, accesibilidad básica y buenas prácticas de marcado.
- **CSS3 y diseño responsive:** Selectores, box model, flexbox, grid, media queries y diseño adaptable a múltiples dispositivos.
- **JavaScript moderno (ES6+):** Variables, funciones, objetos, módulos, asincronía (promesas/async-await) y manipulación del DOM.
- **Frontend con componentes:** Consumo de APIs, manejo de estado e integración de interfaces dinámicas.
- **Backend con Node.js y Express:** Creación de servidores, rutas, middleware y APIs REST.
- **Persistencia y CRUD:** Conexión a base de datos desde Node.js y operaciones CRUD (Crear, Leer, Actualizar, Borrar).
